# To-Do List на Vue 3

Переписывание To-Do List на Vue 3

## Функционал

- Добавление новых задач
- Отметка выполнения (зачёркивание)
- Редактирование задач по двойному клику
- Удаление задач с подтверждением
- Поиск по названию задач
- Переключение светлой/тёмной темы
- Сохранение в localStorage
- Адаптивный дизайн
- Анимации и плавные переходы

## Технологии

- Vue 3 Composition API (через CDN)
- Vanilla CSS с кастомными стилями
- Font Awesome иконки
- LocalStorage для сохранения данных
